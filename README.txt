== Description ==

Configure the way Google Analytics cookies are loaded on your web page.

= Features =

* Personalize the cookies notice and the link to your cookies information page.
* Add easily your Google Analytics code. In addition, you can choose whether Google Analytics cookies are loaded always or only when the cookies notice is accepted

== Installation ==

1. Upload and install Pwg Cookies in the same way you'd install any other plugin.
2. Read the [usage documentation](http://www.programacionwebgalicia.com/pwg-cookies/) on Programación Web Galicia.
